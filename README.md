# Nonogram-Solver;Nonogram自動解題程式
● 什麼是Nonogram：<br>
![image](https://i.imgur.com/F6jycBW.png)<br>
Nonogram是一種圖像遊戲，行列的數字代表圖像中那行(列)黑點的依序數量，<br>
玩家要依靠那些提示畫出實際圖片<br>
<br>
● 程式內容：<br>
研究相關論文中的內容理論實作出解題程式，<br>
並實際測試不同檢查機制的效能差異，<br>
透過修改迴圈和演算法決策提高運算速度<br>
